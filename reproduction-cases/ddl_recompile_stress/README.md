# SQL Stress Test Set

This SQL set is designed to simulate database workload patterns for reliability testing.

## Purpose

The goal is to simulate:

- Complex join queries
- Function execution load
- Procedure execution pressure
- Object invalidation scenarios

## Design

All tables and objects are synthetic.

No production schema or business logic is included.

## Concepts Tested

- Optimizer stress
- Metadata invalidation
- Concurrent execution
- Function call overhead
- Join complexity
